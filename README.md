# Phantom Adaptive Tunnel (PAT)

**Автор:** Kraken992  
**Версия:** 1.0 (2026)

Личная адаптивная система обхода блокировок на базе **sing-box**.

### Основные возможности
- **Primary**: Hysteria 2 (высокая скорость + устойчивость к потерям)
- **Fallback**: VLESS + Reality + XHTTP (максимальная скрытность)
- Автоматическая смена ключей
- Генератор шумового трафика
- Минимальное потребление ресурсов
- Только для личного использования
